🚀 Getting Started

Simply open index.html in any modern web browser. No installation, build process, or server required!

🧪 Quick Tests

Once you open the application, try these quick tests:

  1. Click on "Remote Area Working Acknowledgement" card
  2. Fill in employee name and check the acknowledgment box
  3. Submit the form
  4. Search for the submission by employee name
  5. Click on the submission to view details and export PDF
  6. Try adding a new acknowledgment type
  7. Toggle between light and dark themes

🐛 Troubleshooting

If you encounter issues:

  • Check browser console (F12) for error messages
  • Refresh the page if loading seems stuck
  • Try a different browser if problems persist
  • Debug command: Type debugApp() in console for diagnostics

💡 Tips

  • Use Ctrl/Cmd + D to toggle theme
  • Use Ctrl/Cmd + K to focus search
  • Use Ctrl/Cmd + N to add new type
  • Use Esc to close modals
  • All data is saved locally in your browser
